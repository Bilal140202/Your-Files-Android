## Full Description (4000 chars max)

```
Your Files is a fast, modern, and completely private file manager for Android. No ads, no tracking, no internet — just powerful file management that respects your privacy.

📁 FILE BROWSING & MANAGEMENT
• Browse your entire storage with an intuitive file explorer
• Copy, cut, paste, rename, and move files with ease
• Multi-select for batch operations — delete, move, or share multiple files at once
• Sort by name, size, date, or file type
• Grid and list view options
• Compress files to ZIP and extract ZIP archives

🖼️ BUILT-IN MEDIA VIEWERS
• Image viewer with pinch-to-zoom support
• Video player powered by ExoPlayer for smooth playback
• Audio player for music and recordings
• PDF viewer for documents
• Text and code viewer with syntax-aware display

📊 STORAGE TOOLS
• Storage Analyzer — visualize what's using your space with an interactive donut chart
• Duplicate File Finder — find and remove identical files wasting storage
• Large File Finder — locate the biggest files on your device instantly
• Image Optimizer — reduce image file sizes without noticeable quality loss

🧹 SMART CLEANING
• WhatsApp Cleaner — find and delete WhatsApp media clutter (sent images, videos, voice notes)
• Screenshot Cleaner — quickly find and clean old screenshots
• Recycle Bin — deleted files go to trash first, with one-tap undo restore

🔒 PRIVACY FIRST
• 100% offline — no internet permission, no servers, no cloud
• Zero data collection — no analytics, no tracking, no ads
• All file processing happens entirely on your device
• No account required, no sign-up

🎨 MODERN DESIGN
• Material You (Material 3) design with dynamic theming
• Dark mode support (follows system setting)
• Optimized for phones and tablets
• Clean, fast, and lightweight

Your Files is part of the Your Apps suite — alongside YourNotes and Your Gallery.

Take control of your storage. Download Your Files today.
```
