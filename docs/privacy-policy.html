Privacy Policy

Your Files — File Manager & Storage Cleaner

Last updated: June 2025

Your Files is 100% offline. No data is collected, transmitted, or shared — ever.

1. Overview

Your Files (com.yourfiles.manager) is a file management application developed by Bilal. This privacy policy explains our approach to your data and privacy.

Your Files does not require or use any internet permission. The app operates entirely on your device and has no capability to send or receive data over the internet.

2. Data Collection NONE

Your Files does not collect any data, including but not limited to:

  • Personal information — name, email, phone number, or any identifiers
  • Device information — device ID, IMEI, advertising ID, or any hardware identifiers
  • Usage analytics — no analytics, crash reporters, or telemetry of any kind
  • Location data — no access to GPS or network location
  • App usage data — no tracking of what files you open, how you use the app, or any behavioral data

3. No Ads, No Tracking, No Analytics

Your Files contains no advertisements, no third-party analytics SDKs, and no tracking libraries. There are no ads served through any ad network, and no user behavior is monitored or recorded.

4. On-Device Processing

All file operations performed by Your Files happen entirely on your device:

  • Browsing, copying, moving, renaming, and deleting files
  • Viewing images, videos, audio files, PDFs, and text files
  • Analyzing storage usage and finding duplicate files
  • Optimizing images and cleaning media files

No file content, metadata, or processing results are ever transmitted to any external server.

5. Permissions

Your Files requests the following Android permissions, each with a specific purpose:

  • Storage / All Files Access (MANAGE_EXTERNAL_STORAGE) — Required to browse, manage, and organize files stored on your device. This permission is essential for the app's core file management functionality.
  • Install Packages (REQUEST_INSTALL_PACKAGES) — Required solely to allow users to install APK files from the file browser. No installation happens without explicit user action.

These permissions are used only for their stated purposes. No permission data is collected, read for profiling, or shared externally.

6. No Third-Party Data Sharing

Your Files does not share any data with third parties. Since no data is collected in the first place, there is nothing to share. The app does not integrate with any third-party services, SDKs, or data processors.

7. Data Security

Since all data remains on your device and is never transmitted, the standard Android security model protects your files. Your Files relies on Android's built-in sandboxing and permission system to ensure file operations are authorized.

8. Data Deletion

Since Your Files does not store any user data beyond the Android system's standard app data directory, you can remove all app-related data at any time by:

  • Going to Settings → Apps → Your Files → Storage → Clear Data
  • Uninstalling the app entirely, which removes all app data

Note: Clearing app data or uninstalling the app will not delete your personal files (photos, videos, documents, etc.). It only removes the app's own configuration and cache data.

9. Children's Privacy

Your Files does not collect any information from anyone, including children. The app has no registration, no accounts, and no data collection mechanisms of any kind.

10. Changes to This Policy

If this privacy policy ever changes, the updated version will be published at the same URL and the "Last updated" date will be revised.

11. Contact
